# CxKitty_for_windows
    在Windows上快捷的使用CxKitty
    原项目https://github.com/SocialSisterYi/CxKitty
# 使用项目
    方法一
        点击右上角的code
        下载zip文件
        解压zip文件
        打开文件夹
        双击启动.bat
    方法二
        使用git
            git clone https://github.com/renxinyileng/CxKitty_for_windows.git
            cd CxKitty_for_windows
            双击启动.bat
# 需要修改的项目
    config.yml
    在python-3.10.11-embed-amd64目录下
